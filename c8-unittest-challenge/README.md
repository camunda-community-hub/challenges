# c8-unittest-challenge

![BankOfAn dora_benchmark.png](images/BankOfAndora_benchmark.png)

A process is developped, and the goal is
* to develop some unit test to validate the behavior of the process
* to set up a QA platform, to ensure, day after day, the process react in the same way.

The QA platform must run the unit test every day, or in a CI environment

> The process used is the same as the Load Test challenge

For the challenge, no service task is developped, and it will be simulated.
